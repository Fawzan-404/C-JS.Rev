Data Types

Description (English):

Defines the type of data a variable can hold. Common types include int, float, and char.

Description (Roman Urdu):

Data types variable ke data ka type define karte hain. Aam data types mein int, float, aur char shamil hain.

Examples:

int age = 20; // Integer type
float height = 5.9; // Floating-point type
char grade = 'A'; // Character type

Keywords

Description (English):

Keywords in C are reserved words that have special meaning. Examples include int, return, if, and while.

Description (Roman Urdu):

C mein, keywords woh reserved words hote hain jo special meaning rakhte hain. Misal ke taur par int, return, if, aur while.

Examples:

int main() {
  return 0;
}

Strings

Description (English):

In C, strings are arrays of characters ending with a null character '\0'. They are used to represent text.

Description (Roman Urdu):

C mein, strings characters ka array hoti hain jo null character '\0' par end hoti hain. Ye text ko represent karne ke liye use hoti hain.

Examples:

char name[20] = "John Doe"; // String example
printf("Name: %s", name); // Prints the string

Arithmetic Operators

Description (English):

Arithmetic operators in C are used to perform basic arithmetic operations. They include + (addition), - (subtraction), * (multiplication), / (division), and % (modulus).

Description (Roman Urdu):

C mein arithmetic operators basic arithmetic operations perform karte hain. Ye + (addition), - (subtraction), * (multiplication), / (division), aur % (modulus) par mushtamil hote hain.

Examples:

int a = 10, b = 5;
int sum = a + b; // Addition
int difference = a - b; // Subtraction
int product = a * b; // Multiplication
int quotient = a / b; // Division
int remainder = a % b; // Modulus

Format Specifiers

Description (English):

Format specifiers in C define the type of data that a variable holds and how it should be printed. Examples include %d for integers, %f for floats, and %c for characters.

Description (Roman Urdu):

Format specifiers C mein us data type ko define karte hain jo variable hold karta hai aur kaise print hona chahiye. Misal ke taur par %d integers ke liye, %f floats ke liye, aur %c characters ke liye use hota hai.

Examples:

int num = 10;
printf("Number: %d", num); // Outputs: Number: 10

Control Statements

Description (English):

Control statements in C alter the flow of program execution based on conditions or loops. These include if, else, for, while, and switch.

Description (Roman Urdu):

C mein control statements program ke execution flow ko conditions ya loops ki buniyad par alter karte hain. In mein if, else, for, while, aur switch shamil hain.

Examples:

int a = 10;
if (a > 5) {
  printf("a is greater than 5");
} else {
  printf("a is less than or equal to 5");
}

Functions

Description (English):

Functions in C are blocks of code designed to perform a particular task. They help in modular programming and reusability. A typical function includes a return type, name, parameters, and body.

Description (Roman Urdu):

C mein functions code ke blocks hote hain jo kisi khas task ko perform karne ke liye design kiye gaye hain. Ye modular programming aur reusability mein madad karte hain. Aik typical function return type, name, parameters, aur body par mushtamil hota hai.

Examples:

int add(int x, int y) {
  return x + y;
}

Pointers

Description (English):

Pointers in C are variables that store memory addresses of other variables. They are powerful but can be complex to use correctly. Commonly used with arrays, functions, and dynamic memory allocation.

Description (Roman Urdu):

C mein pointers aise variables hote hain jo doosray variables ke memory addresses ko store karte hain. Ye powerful hote hain lekin sahi tarah se use karna complex ho sakta hai. Ye aksar arrays, functions, aur dynamic memory allocation ke saath use hote hain.

Examples:

int x = 10;
int *ptr = &x;
printf("Value of x: %d", *ptr); // Outputs: Value of x: 10

Arrays

Description (English):

Arrays in C are collections of variables of the same type, stored sequentially in memory. They are used to store multiple values in a single variable.

Description (Roman Urdu):

C mein arrays aise variables ki collections hoti hain jo same type ke hote hain aur memory mein sequentially store hote hain. Ye aik single variable mein multiple values ko store karne ke liye use hote hain.

Examples:

int numbers[5] = {1, 2, 3, 4, 5};
printf("First element: %d", numbers[0]); // Outputs: First element: 1

Structures

Description (English):

Structures in C are user-defined data types that allow grouping of different data types under a single name. They are used to model real-world entities.

Description (Roman Urdu):

C mein structures user-defined data types hote hain jo different data types ko aik single name ke under group karte hain. Ye real-world entities ko model karne ke liye use hote hain.

Examples:

struct Person {
  char name[50];
  int age;
  float height;
};
struct Person john = {"John Doe", 30, 5.9};
printf("Name: %s, Age: %d, Height: %.1f", john.name, john.age, john.height);

C Token

Description (English):

Token is the smallest indivisual unit in a program , Tokens are the building blocks of a C program and are used to construct the entire code. These are six types of tokens in C

Description (Roman Urdu):

Tokens C program ke chhote aur indivisual units hote hain. Ye program ke har hissay ko banane ke liye use kiye jaate hain. Tokens C program ki building blocks hain aur inki madad se poora code construct kiya jaata hai. C mein 6 tarah ke tokens hote hain:

Keywords

Keywords: Reserved words that have a special meaning in C. Jaise - auto , int , else , while .... etc.

Identifiers

Identifiers: Names given to variables , functions , arrays and other user-defined items. Jaise - name , age , address .... etc.

Rules:

[1]Must began with a letter [A-Z] , [a-z] and '_'

[2]Can contain letters , digits[0-9] and underscores '_'

[3]Cannot contain spaces , special characters and symbols

[4]Cannot be a keyword , Case-Sensitive ex- variable and Variable are different

Constants

Constants: Fixed values that do not change during the execution of a program

Types:

[1]Integer constant (100,-20)

[2]Floating point constant (eg: 3.14, -0.001)

[3]Character constants (eg: 'A','Z')

[4]String constants (eg: "Hello, World!", "C Programming")

Operators

Operators: Symbols used to perform operations on operands

Types:

[1]Arithmatic operator ('+','-','*','/','%')

[2]Relational Operators (==, !=, >, <, >=, <=)

[3]Logical Operators (&&, ||, !)

[4]Bitwise Operators (&, |, ^, ~, <<, >>)

[5]Assignment Operators (=, +=, -=, *=, /=, %=, <<=,)
